Нажмите Win + R, введите taskschd.msc и нажмите Enter
В правой панели нажмите "Создать задачу..."
Вкладка "Общие":
Имя: Очистка кеша приложений
Выберите "Выполнять независимо от регистрации пользователя"
Отметьте "Выполнять с наивысшими правами"
В настройках выберите вашу ОС
Вкладка "Триггеры":
Нажмите "Создать..."
Начать задачу: "При запуске системы"
Дополнительно можно добавить задержку: 1 минута
Нажмите ОК
Вкладка "Действия":
Нажмите "Создать..."
Действие: "Запуск программы"
Программа: C:\Windows\System32\cmd.exe
Аргументы: /c "C:\Scripts\ClearCache.bat"
Нажмите ОК
Вкладка "Условия":
Снимите галочку "Запускать только при питании от сети"
Остальное можно оставить по умолчанию
Вкладка "Параметры":
Отметьте "Выполнять пропущенную задачу как можно быстрее..."
Нажмите ОК для создания задачи
